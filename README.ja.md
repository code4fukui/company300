# はばたく中小企業・小規模事業者300社

中小企業庁の「2023年度 はばたく中小企業・小規模事業者300社」の情報をもとに、日本地図上で企業数を視覚化したダッシュボードを提供するプロジェクトです。

## デモ
https://code4fukui.github.io/company300/

## 機能
- 日本地図上に各都道府県の企業数を表示
- 都道府県をクリックすると、その都道府県の企業一覧を表示

## 必要環境
特に必要ありません。ウェブブラウザで閲覧できます。

## 使い方
特に操作は必要ありません。地図を眺めるだけで企業数の分布が確認できます。

## データ・API
データは中小企業庁の「2023年度 はばたく中小企業・小規模事業者300社」の情報を基に作成しています。
