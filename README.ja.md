# company300

2023年度に選定された「はばたく中小企業・小規模事業者300社」を可視化するインタラクティブなダッシュボードです。このツールにより、すべての都道府県における選定企業の地理的な分布を把握できます。

## 機能

- **インタラクティブマップ:** 都道府県ごとの選定企業の密度を示すために色分けされた、日本のTabular Map（表形式マップ）。
- **詳細一覧:** 任意の都道府県をクリックすると、選定分野（カテゴリ）や公式サイトへのリンクを含む企業一覧を表示します。
- **都道府県ランキング:** 選定企業数に基づいて都道府県が自動的にランク付けされ、その順位がマップ上に直接表示されます。

## 使い方

最新のWebブラウザで `index.html` を開いてください。インストールやビルドの手順は不要です。

## データ

本アプリケーションは `company300.csv` に保存された企業データを使用しています。このデータは、経済産業省（METI）が公表している公式リストからスクレイピングしたものです。

- **出典:** [2023年度 はばたく中小企業・小規模事業者300社 (METI)](https://www.chusho.meti.go.jp/keiei/sapoin/monozukuri300sha/2023/detail.html)

## 開発

スクレイピングスクリプトを実行することで、データを更新できます。これには [Deno](https://deno.land/) がインストールされている必要があります。

データを再スクレイピングして `company300.csv` を更新するには、プロジェクトのルートディレクトリから以下のコマンドを実行します。

```sh
deno run --allow-net --allow-read --allow-write scrape.js
```
